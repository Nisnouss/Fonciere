# Fonciere

## Objectif

Créer un site en utilisant HTML et CSS conformément au modèle d'un site immobilier.

## Composantes

### Header

Le Header contient le logo avec un menu cliquable (Rueil malmaison, Bureau à meaux, Bureaux à louer, Galerie, Contact).

### Main

Le main est composé de six sections toutes référencées dans le Header.

### Footer

Le footer est composé du copyright et des mentions légales.
